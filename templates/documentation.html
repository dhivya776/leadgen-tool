LeadGen Pro - Documentation

Welcome to LeadGen Pro! This app allows users to filter companies based on industry, location, employee count, and revenue to generate potential leads efficiently.

Features

  • Search companies by industry and location.
  • Filter results by minimum and maximum number of employees.
  • Filter results by revenue range in millions of dollars.
  • View company contact emails directly.
  • Clean, simple user interface for easy navigation.

How to Use

  1. Enter your desired industry (e.g., Software, Healthcare).
  2. Specify location (city, state).
  3. Adjust employee count and revenue filters as needed.
  4. Click “Find Companies” to see filtered results.
  5. Click email addresses to contact companies directly.

Technology Stack

This project is built with:

  • Flask - Python web framework
  • HTML, CSS, JavaScript - Frontend
  • Fetch API - For dynamic search requests
